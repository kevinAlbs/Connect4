Connect Four API Endpoint

The source and documentation is on the GitHub repository

There is currently one functional endpoint (more to come). All can be used as either GET or POST requests.

http://kevinalbs.com/connect4/back-end/index.php/getMoves

This is the endpoint for the main AI. It requires one input, a completed board, which must be of the following form:

  • A single string containing 49 characters, all mapping to locations on the 7 x 7 connect four board (going left to right, top to bottom)
  • Each character must be either 0 (for empty), 1 (for player 1), or 2 (for player 2)

There is an additional optional parameter for specifiying which player to check for moves. The default is to check for player 1.

The endpoint returns a JSON object. It contains the column of the move (zero-based, so 0 is the left-most column and 6 is the right-most) mapping to it's relative min-max score

Example curl request:

            curl 'http://kevinalbs.com/connect4/back-end/index.php/getMoves?board_data=0000000000000000020000001200000210000021001012100&player=2'
